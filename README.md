# بوت بايثون

مشروع بوت تيليجرام احترافي جاهز للرفع إلى GitHub.

## المميزات
- لوحة إدارة عبر /admin
- تعديل اسم المتجر ورسالة الترحيب
- تعديل نص التواصل وحول المتجر
- إدارة الخدمات: إضافة، تعديل، حذف وعرض
- تعديل أسماء الأزرار الأساسية
- تغيير ألوان الأزرار
- إخفاء وإظهار الأزرار
- تغيير ترتيب الأزرار
- استقبال طلبات العملاء
- تصدير الإعدادات

## التشغيل على الهاتف
```bash
pip install -r requirements.txt
python main.py
```

## Railway
أضف المتغيرات:
- BOT_TOKEN
- ADMIN_ID

ثم اجعل أمر التشغيل:
```bash
python main.py
```
